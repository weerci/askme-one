Вкладка "Соединение"

 Top  Previous  Next

 

На вкладке "Соединения" задаются параметры подключения к базе данных.

 

Возможны два типа соединения:

1.С использованием файла настроек tnsnames.ora клиента Oracle. Для настройки такого соединения необходимо соответствующим образом отредактировать файл настроек.
2.ODBC - соединение. Используется драйвер ODBC.

 

Кнопки "Тест соединения", позволяют проверить правильность внесенных настроек.

Если соединение с базой данных прошло успешно - появиться сообщение: "Соединение успешно установлено".

В противном случае появится сообщение, содержащие описание возникшей ошибки.
