Вкладка "Константы"

 Top  Previous  Next

 

Влкадка "Константы" состоит из элементов, сгруппированных в двух областях - "Получить файл" и "Переменные".

 

Получить файл

Здесь можно получить файл отчетности на заданное число.

 

"Получение отчета на дату" - поле, содержее значение даты, на которое получается отчет.

 

"Отправить отчет по e-mail" - если флажок установлен, то полученный файл будет отправлен на адресс электронный почты, указанный в настройках влкадки "Настройки".

 

Кнопка "Получить отчет" - кнопка, нажатие на которую инициирует процесс получения отчетности.
Поскольку запрос может выполнятся долго, то его выполнение запускается в отдельном потоке. При этом кнопка становиться недоступной, до завершения выполнения запроса. Ход выполения запроса отображается в строке состояния.

 

Кнопка "Просмотр" - после отработки запроса данные помещаются в файл, содержимое которого можно просмотреть нажав на эту кнопку.
Важно!

1.До тех пор, пока новый запрос полностью не отработает, при нажатии на эту кнопку отображается содержимое предыдущего запроса.
2.Если выбран флаг "Отправить отчет по e-mail", то содержимое файла архивируется и данные отправляются на выбранный электронный адрес, поэтому после выполнения запроса они не доступны при нажатии на кнопку "Просмотр". Архив отправленных zip-файлов храниться по пути "Каталог установки/resultFile"

 

Переменные

Поля размещенные в этой области группировки, содержат значение одноименных аттрибутов выходного файла XML.

 

"Файл настроек Codes_base.xlsx" - при нажатии на эту кнопку, открывается файл настроек программы Codes_base.xlsx. Из этого файла беруться данные для формирования запроса к базе и наименования точек доступа.

После открытия, редатирования и сохранения отредактированных данных средствами Excel, они становяться доступны для формирования отчетности сразу же, т.е. - перегружать программу или перезапускать сервер автоматической загрузки нет необходимости.
