Описание программы

 Top  Next

Программа предназначена для получения данных из базы, в соответствии с предварительно подготовленным текстом запроса, конвертации полученных данных в файл XML, и отправки отчета

с вложением полученного файла по e-mail.

 

В соответствии с выполняемыми задачами, функционал программы делиться на следующие части:

1.Подготовка текста SQL-запроса к базе данных.
2.Настройка соединения с базой.
3.Настройка параметров отправки почтовых сообщений.
4.Настройка расписания запуска программы на выполение

 

Особенности реализации

Программа реализованна в виде фонового процесса. Это означает, что при запуске программы на выполнение, интерфейс основного окна программы не отображается, а отображается информационная иконка в трее (системный лоток).

Доступ к интерфейсу программы можно получить дважды кликнув по этой иконке, или выбрав пункт "Настройки", выпадающего меню, появляющегося при клике правой кнопкой мышки, по ней же.

Программа не завершает свою работу, при закрытии ее по "крестику" в правом верхнем углу формы. Вместо завершения - она сворачивается в трей. Для завершения выполнения программы нужно выбрать пункт "Выход", выпадающего меню.

 

Режимы получения отчетности

Получение отчета возможно двумя способами: в автоматическом режиме и по требованию пользователя.

В автоматическом режиме отчет подготавливается один раз в сутки, в определенный период времени, предварительно указанный пользователем в настройках программы. Работа программы в этом режиме обеспечивается сервером автоматической загрузки, который пользователь должен явно запустить, после установки нужных настроек программы.

Режим по требованию позволяет пользователю сформировать отчет в любой момент времени.

Оба режима являются независимыми, т.е. пользователь может получать отчетность, на интересующую его дату в режиме "По требованию", при запущенном автоматическом режиме.

 

Сохранение данных

Настройки программы и данные отчетнотсти, подготовленные пользователем, сохраняются в файле настройки программы "sysprop.dmp". После сохранения внесенных изменений (нажатие на кнопку "Сохаранить"), новые данные применяются сразу же, к обоим режимам, без перезагрузки программы или перезапуска сервера автоматической загрузки.

XML-файлы отправленные по e-mail, сохраняются в папке "resultFile", каталога установки программы.

 

Ошибки выполнения

Поскольку программа предназначена для выполнения в фоновом режиме, то никаких пользовательских информационных сообщений, при работе программы, не предусмотренно. Вся информация о ходе или возникших проблемах выполнеия записывается в лог-файл программы "Askme.log", расположенный в директории установки. Для удобства доступа к лог-файлу, на вкладке настройка имеется кнопка "Лог-файл".

Текущее состояние сервера автоматической загрузки (ожидание, выполнение SQL-запроса, отправка e-mail и т.д.) отображается в левой нижней части формы.
