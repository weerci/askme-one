Настройки

 Top  Previous  Next

Основная форма программы содержит четыре закладки, с различными настройками:

1.Константы - на этой закладке определяются константы, которыми будут инициализированны переменные в итоговом файле XML.
Кроме того, с этой влкадки можно получить файл отчета в режиме "По требованию"
2.Запрос - на этой вкладе расположено тектовое поле, в котором редактируется текст SQL-запроса к базе данных.
3.Соединение - на владке расположены поля для задания условий коннекции к базе данных.
Возможны два варианта доступа к данным: tnsnames - соединение через драйвер Oracle и ODBC - соединение
4.Настройки - вкладка содержит параметры настройки таймера, e-mail сообщений, кнопку запуска режима автоматической загрузки и доступка к лог-фалу.

 

 

При изменении значений полей на любой из вкладок, становиться активной кнопка "Сохранить". При нажатии на эту кнопку новые данные сохраняются в файле настройки программы. При нажатии на кнопку "Отмена" - внесенные изменнеия отменяются.

 

В левой нижней части формы расположена строка состояния сервера. В этой строке отображается текущее состояние получения данных, возможны следующие состояния:

•Подготовка запроса
•Выполнение запроса
•Формирование XML
•Отправка e-mail
•Ожидание
