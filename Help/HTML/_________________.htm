Вкладка "Настройки"

 Top  Previous 

 

На вкладке "Настройки", содержаться установки время срабатывания таймера, настройки e-mail, кнопки запуска/остановки сервера и доступа к лог-файлу.

 

Установки таймера

Здесь задаются установки для автоматического запуска программы.

 

Поле "Промежуток времени..."

В двух полях указывается промежуток времени, в течении которого сервер будет пытаться получить запрос и отправить данные. Во всякий другой промежуток времени сервер не будет пытаться установить соединение с базой.

 

Поле "Интервал срабатывания..."

Содержит значение в минутах, по истечении которого, сервер будет проверять - не выполнились ли условия для формирования очередного запроса.

Важно!

1.Значение в этом поле должно быть меньше, чем разница значений в полях "Промежуток времени", поскольку при большем значении может оказаться так, что сервер пропустит время формирования отчетности.
2.Поле "Промежуток времени" не должен содержать внутри себя полночь.

 

Настройки e-mail

Настройки smtp-сервера, с которого будет отправляться почта.

 

Запуск сервера и лог-файл

При запуске программы сервер автоматического запуска не активен. Это связанно с предоставлением пользователю возможности предварительной настройки сервера (константы, запрос, адрес e-mail и т.д.).
Для запуска сервера нужно нажать кнопку "Старт". При этом статус сервера поменяется на "Сервер запущен", а название кнопки на "Стоп".

Для остановки сервера нужно нажать кнопку "Стоп". При этом статус сервера поменяется на "В настоящий момент сервер остановлен", а название кнопки поменяется на "Старт".

 

События возникающие в ходе выполнения программы и возникшие ошибки сохраняются в лог-файле. Просмотреть лог-файл можно нажав на кнопку "Лог-файл". Сообщения в файл записываются последоватльно, по мере их поступления, поэтому более поздние сообщения храняться ближе к концу файла.
