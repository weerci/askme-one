Вкладка "Запрос"

 Top  Previous  Next

 

Вкладка запрос содержит текстовое поле в котором можно отредактировать текст SQL-запроса.

 

Текст запроса содержит два подстановочных значения {0} и {1}.

 

Первое подстановочное значение - содержит данные поля ID_Object файла Codes_base.xlsx, в формате: "981002, 983037, ..., 981008". Данные берутся из  файла, имющиеся дубликаты удаляются и формируется указанная последовательность.

Для редактирования этих данных используются возможности программы Excel.

 

Второе подстановочное значение - содержит дату, на которую формируется запрос.

Важно!

1.В автоматическом режиме, значение содержит текущую дату.
2.В режиме получения отчетности по требованию, подставляется значение из поля "Получение отчета на дату" вкладки "Константы"
